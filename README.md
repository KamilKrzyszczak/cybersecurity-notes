# Cybersecurity Notes

A structured knowledge base covering networking, web security, common vulnerabilities, tools, and pentesting methodology.

This repository documents my learning process and practical understanding of offensive security fundamentals.

---

## Structure

### 01 - Networking
Core networking concepts required for security work:
- OSI model
- TCP vs UDP
- Ports
- DNS
- TLS / HTTPS

### 02 - Web Fundamentals
How modern web applications work:
- HTTP (methods, headers, versions)
- Authentication mechanisms
- Sessions & cookies
- JWT
- Browser security model (SOP, CORS, storage)

### 03 - Vulnerabilities
Common web vulnerabilities and attack patterns:
- IDOR
- SQL Injection
- XSS
- Path Traversal
- File Upload
- Business Logic flaws
- Open Redirect

Each note focuses on:
- How it works
- Why it is vulnerable
- How it is exploited
- What to look for during testing

### 04 - Tools
Security tools used in practice:
- Burp Suite
- curl
- nmap
- Linux basics

### 05 - Methodology
Practical approach to web application testing:
- Recon checklist
- Web testing process
- Pentest flow

---

## Goal

To build a clean, structured, and interview-ready knowledge base that reflects practical understanding of web security and pentesting fundamentals.
